This is a privacy oriented and deblobbed web browser based on Mozilla technology. It enables many features upstreamed by the [https://wiki.mozilla.org/Security/Tor_Uplift Tor Uplift project] using preferences from the [https://github.com/arkenfox/user.js arkenfox-user.js project]. It is compiled from source and proprietary blobs are removed using scripts by Relan from [https://gitlab.com/relan/fennecbuild here]. It is recommended to install 'uBlock Origin'.
